# node-terminus

Comando para sincronizar os dados dos episódios

```sh
$ npm run sinc
```

Comando para gerar os CSV's com os dados sincronizados

```sh
$ npm run csv
```

## Pré-Requisitos

Ter instalado em sua máquina:

- _NodeJS_
- _MongoDB_

Na pasta do projeto abra o terminal e digite:

```sh
$ npm install
```
